# Build Express

[GO BACK](../index.md)

1. Install npm

   > npm init

2. Install express

   > npm install express

3. create a js file, example app.js

4. boiler plate for the js file made app.js

Express boiler plate with end points [Click here](../boiler-plate/express-boiler-plate.md)

5. Run app.js

   > node app.js

   You need to stop and run again app.js if you modify the file

6. Install nodemon for running app.js
   No need to stop and start to see changes in app.js

   > npm install nodemon

   to use:

   > npx nodemon app.js
